# Macro Trackr

A nutrition tracking application with JWT authentication.

## Features

- Track daily macros (protein, carbs, fats)
- User authentication
- Data visualization
- CSV export

## Installation

```bash
bun install
```

## Scripts

| Command | Description |
|---------|-------------|
| `bun run dev` | Run both frontend and backend in development |
| `bun run dev:backend` | Run only the backend in development |
| `bun run dev:frontend` | Run only the frontend in development |
| `bun run build` | Build the frontend for production |
| `bun run build:backend` | Build the backend for production |
| `bun run start` | Start the production server |
| `bun run test` | Run frontend tests |
| `bun run lint` | Run frontend linter |
| `bun run typecheck` | Run TypeScript type checking |

## Running

```bash
bun run dev
```
